Merge sort divide the list into half. Then, the half is subdivided again until there is only 1 element in the pile. Then, merge them together. A temporary array is created when merging. Compare first element of both pile and insert the smaller value to the temporary array. Next, proceed to compare the next element and compare again. After all values have been inserted.
